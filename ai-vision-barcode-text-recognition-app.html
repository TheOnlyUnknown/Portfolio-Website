AI-Powered Vision / Barcode & Text Recognition App

Android app using ML Kit for barcode scanning, text recognition, image labelling, and Firebase integration.

Overview

Turning camera input into useful information.

This project is an Android application that uses ML Kit to perform barcode scanning, text recognition, and image labelling. It combines mobile development with applied AI and stores processed results in Firebase for a complete end-to-end user workflow.

Goal

Build a mobile application that can capture visual data, extract meaningful information in real time, and store results for later use.

Methods

Developed the app in Android Studio, integrated ML Kit vision APIs for multiple recognition tasks, designed interactive UI flows, and connected Firebase Realtime Database for storing and managing outputs.

Tools

Android Studio, Java/Kotlin, Google ML Kit, Firebase Realtime Database, XML UI design, and mobile app architecture principles.

What I Worked On

  • Integrated ML Kit features for barcode, text, and image recognition.
  • Android app screens and user flows for image-based recognition tasks.
  • Integrated ML Kit for barcode scanning, text recognition, and image labelling.
  • Enabled users to view, edit, and manage detected results inside the app.
  • Connected the application to Firebase for storing recognition outputs in real time.
  • Combined AI functionality with practical mobile app design for a usable end product.
