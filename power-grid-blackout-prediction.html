Power Grid Blackout Prediction

Machine learning model for identifying blackout risk patterns from power system data.

Overview

Predicting risk before outages happen.

This project applies machine learning to predict blackout risk using power grid operational data. It follows a full ML workflow from preprocessing and imbalance handling to model training, evaluation, and interpretation, with the goal of supporting earlier and better decision-making.

Goal

Develop a machine learning model that identifies blackout risk from power grid operating data and helps distinguish unstable conditions from normal system behaviour.

Methods

Cleaned and prepared generator and bus data, handled class imbalance, engineered useful features, and trained classification models including logistic regression, random forest, and XGBoost for comparison.

Tools

R, Python, pandas, scikit-learn, caret, ROSE, XGBoost, ggplot2, and performance metrics such as accuracy, precision, recall, F1-score, and ROC-AUC.

What I Worked On

  • Preprocessed power grid data from multiple sources for machine learning analysis.
  • Handled imbalanced classes using resampling techniques like ROSE and SMOTE.
  • Trained and compared multiple classification models for blackout prediction.
  • Evaluated model performance using confusion matrices, ROC analysis, and feature importance.
  • Interpreted results to show which grid indicators contributed most to blackout risk.
