Stock Market Movement Prediction

Time-series machine learning project for predicting next-day stock movement from historical market data.

Overview

Reading market movement through data.

This project applies machine learning to forecast next-day stock price direction using historical market data. It includes data preprocessing, feature engineering, model comparison, and time-series validation to build a practical workflow for financial prediction.

Goal

Build a machine learning model that predicts whether a stock will move up or down based on past price behaviour and derived market features.

Methods

Prepared historical stock data, engineered lag-based and return-based features, and trained multiple models including logistic regression, SVM, and random forest using time-series cross-validation.

Tools

Python, pandas, NumPy, scikit-learn, matplotlib, seaborn, TimeSeriesSplit, and feature engineering techniques for financial data.

What I Worked On

  • Cleaned and organised multi-company historical stock price data.
  • Predictive features such as returns, rolling trends, and lag variables.
  • Trained and compared multiple machine learning models for next-day movement prediction.
  • Evaluated model performance using accuracy, F1-score, ROC-AUC, and cross-validation.
  • Analysed results to understand which modelling approach worked best for time-series data.
